githubPage 配置域名

Apr 21, 2020杂谈点击

买域名

国内购买域名可在阿里云上购买

域名解析

之后需要在阿里云控制台操作域名，对域名进行解析操作。选择想到操作的域名，如
点击解析设置，跳转页面。点击新手引导，在弹窗输入githubname.github.io的ip。

ip 获取方式

在电脑CMD 输入，ping githubName.github.io,变可得到

github 设置

打开githubpage的项目-setting -拉后面-在custom domain 输入域名

以下配置等同上方github仓库配置

若是为了配置hexo 博客的域名，可在source下创建CNAME文件，并在文件内写入域名即可。（在source上创建文件，在每次生成文件，部署到github后都会在根目录）
