类型和值

May 24, 2019JavaScript点击

类型

概念：对语言引擎和开发者来说，类型是值得内部特征，它定义了值得行为，以使其区别于其他值

内置类型

  • null 空值
  • undeined 未定义
  • boolean 布尔值
  • number 数字
  • string 字符串
  • symbol 符号
  • object 对象

注：除了对象外，其他都是基本类型。使用 typeof 检查数据类型时，注意 typeof null == object。

所以我们需要使用复合条件来检测 null类型

值

数组

需要注意的点：

  1. 创建“稀疏”数组时，空白单元会被赋值为 undefined

  2. 数组通过数字进行索引，但也可以包含字符串键值和属性（但是这些并不计算在数组长度内）

  3. 如果字符串键值能够被强制转换为十进制数字的话，它就会被当做数字索引来处理。

类数组
arguments 对象是一个类数组，通常通过 indexOf（），concat（），forEach（） 来将类数组转换成真正的数组；

字符串

字符串也是一个类数组数据，它具备着 length ，indexOf（）等方法。

数字

JavaScript 中的数字类型是基于 IEEE 754 标准来实现的，该标准通常也被称为 “浮点数”。javascript使用的是“双精度”格式。（64位的二进制）

  1. 特别大和特别小的数字默认用指数格式表示。与 toExponential（）函数的输出结果相同

  2. 数字值可以使用 Number 对象进行封装，所以可以使用 Number.prototype 中的方法。

    1. tofixed() 指定小数部分的显示位数；
    2. toPrecision() 指定有效数位的显示位数；

  3. 0.1 + 0.2 ！== 0.3

    1. 通过设置一个误差范围值，来判断左右两边是否相等。通常称为“机器精度”，对于 JavaScript 来说，这个值是2^-52（2.220446049250313e-16）。从 es6 开始，该值定义在 Number.EPSILON 中。

  4. 最大的浮点数大约是 1.798e+308，它定义在 Number.MAX_VALUE 中。

  5. 最小的浮点数定义在 Number.MIN_VALUE 中，大约是5e-325，他不是负数，但无限接近于0！
  6. 能够被安全呈现的最大整数是 2^53 - 1，即9007199254740991，被定义为 Number.MAX_SAFE_INTEGER.
  7. 最小的整数是-9007199254740991，被定义为 Number.MIN_SAFE_INTEGER。

  8. 整数检测 Number.isInteger()

  9. 检测是否是安全的整数。 Number.isSafeInteger()

  10. setTimeout() 函数返回一个数值（计时器间隔的唯一标识符，用来取消计时）
  11. NAN 不是数字的数字，仍然是数字类型，是唯一一个非自反的值。使用 isNaN（）函数来判断是否是 NaN
  12. 零值，JSON.stringify(-0) 返回 “0”，JSON.parse(“-0”) 返回-0。
  13. 判断 -0，使用isNegZero（）
  14. 特殊等式：es6 新加入一个工具方法 Object.is() 来判断两个值是否绝对相等。

注：var a = 42. 这种表示方式是可以的。

值和引用

  1. 简单值总是通过值复制的方式来赋值/传递，包括null，undefined，bollean，number，string，symbol
  2. 复合值–对象，通过引用复制的方式来赋值/传递值，包括数组，对象
